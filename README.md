## 🚀 Overview

**Comproller** is a high-performance HR and Payroll management suite built with the latest Laravel ecosystem. It provides a seamless experience for managing employees, attendance, salaries, and document generation.

## ✨ Key Features

- 👥 **Advanced Employee Management**: Detailed profiles including personal, financial, and contact data.
- 💰 **Payroll System**: Automated payroll generation and management.
- 📅 **Attendance & Leave Tracking**: Keep track of employee hours and leave requests with FullCalendar integration.
- 🔐 **Enhanced Security**: Built-in 2FA (TOTP) for secure administrative access.
- 📄 **Document Generation**: Bulk ID card printing, contract generation, and medical referral downloads via PDF.
- 📊 **Dynamic Dashboard**: Real-time insights into your organization's workforce.

## 🛠️ Tech Stack

- **Framework**: [Laravel 12+](https://laravel.com)
- **Admin Panel**: [Filament v5](https://filamentphp.com)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com)
- **Frontend**: [Livewire](https://livewire.laravel.com)
- **Database**: PostgreSQL / MySQL / SQLite
- **PDF Engine**: [Laravel DOMPDF](https://github.com/barryvdh/laravel-dompdf)

## 📦 Installation

To get started with Comproller, run the following commands:

```bash
# Clone the repository
git clone https://github.com/ItzKssBerc/comproller.git
cd comproller

# Automated setup (installs deps, migrates db, builds assets)
composer setup
```

## 👨‍💻 Development

Start the development servers:

```bash
composer dev
```
