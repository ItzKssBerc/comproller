## 🚀 Overview

**Comproller** is a high-performance HR and Payroll management suite built with the latest Laravel ecosystem. It's designed to replace legacy, clunky ERP systems with a **premium, smooth, and animated user experience**. 

Featuring the unique **"Aurora"** animated gradient background and a sophisticated **"Typewriter"** entrance effect, Comproller proves that enterprise software can be beautiful.

## ✨ Key Features

- 👥 **Advanced Employee Management**: Full 360° view of staff including personal, financial, and contact data.
- 💰 **Payroll System**: Automated payroll generation with high accuracy.
- 📅 **Attendance & Leave Tracking**: Seamless calendar integration with **FullCalendar**.
- 🔐 **Zero-Trust Security**: Multi-Factor Authentication (MFA) via Google Authenticator built-in.
- 📄 **Smart Documents**: Bulk PDF generation for ID cards, contracts, and medical referrals.
- 🌈 **Modern UI**: Powered by Tailwind CSS v4, featuring glassmorphism and smooth animations.

## 🛠️ Tech Stack

- **Core**: [Laravel 12+](https://laravel.com) & [PHP 8.5+](https://php.net)
- **Admin Engine**: [Filament v5](https://filamentphp.com)
- **Real-time UX**: [Livewire](https://livewire.laravel.com)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com)
- **Database**: PostgreSQL / MySQL / SQLite
- **PDF Engine**: [Laravel DOMPDF](https://github.com/barryvdh/laravel-dompdf)

## 📦 Installation & Quick Start

Setting up Comproller is a breeze thanks to the automated setup script:

```bash
# 1. Clone the repository
git clone https://github.com/ItzKssBerc/comproller.git
cd comproller

# 2. Run the automated setup
# This installs dependencies, sets up the database, and builds assets.
composer setup
```

## 👨‍💻 Development

To start the local development environment:

```bash
composer dev
```
